### 2. **API接口封装** (`src/api/`)

#### **用户接口** (`userApi.js`)
- `login()` - 用户登录
- `register()` - 用户注册
- `checkUsername()` - 检查用户名是否存在

#### **题目接口** (`questionApi.js`)
- `getQuestionList(subjectType)` - 根据题型查询题目列表
- `getRandomQuestions(subjectType, count)` - 随机获取题目
- `getQuestionDetail(id)` - 查询题目详情

#### **记录接口** (`recordApi.js`)
- **答题记录**：
  - `saveAnswerRecord()` - 保存答题记录
  - `updateAnswerRecord()` - 更新答题记录
  - `getAnswerPage()` - 分页查询答题记录
  - `getWrongQuestions()` - 获取错题列表
  - `getCorrectRate()` - 获取正确率统计
  - `getExamStats()` - 获取考试统计
- **背题进度**：
  - `getBackProgress()` - 获取背题进度
  - `saveBackProgress()` - 保存背题进度
- **笔记**：
  - `saveNote()` - 保存笔记
  - `getNote()` - 获取笔记
  - `getAllNotes()` - 获取所有笔记
- **AI接口**：
  - `ask()` - AI问答（70秒超时）

**特殊处理**：
- ✅ 自动初始化用户ID（存储在localStorage）
- ✅ 请求头自动携带 `X-User-Id`
- ✅ 参数类型自动转换（数字、布尔值）
- ✅ AI接口使用独立请求实例（70秒超时）

### 4. **AI字幕实现** (`useWhisperSubtitle.js`)

**技术原理**：
1. 使用Web Audio API捕获视频音频流
2. 重采样到16kHz单声道（Whisper要求）
3. 使用Whisper.cpp WASM模型进行实时语音识别
4. 将识别结果实时显示为字幕

**关键代码流程**：
```javascript
// 1. 加载Whisper WASM模块
loadWhisper() → 加载libmain.js → 初始化模型

// 2. 创建音频上下文
AudioContext (16kHz采样率) → MediaElementSource

// 3. 音频处理
ScriptProcessor → 重采样 → recognize() → 显示字幕
```

**模型文件**：
- `/public/whisper/ggml-tiny.bin` - Whisper模型
- `/public/whisper/libmain.js` - WASM加载脚本
- `/public/whisper/libmain.wasm` - WASM二进制文件

---

### 5. **用户状态管理**

**存储方式**：localStorage

**存储内容**：
- `userInfo` - 用户信息（登录后存储）
- `userId` - 用户ID（自动生成，格式：`user_${timestamp}_${random}`）

**初始化逻辑**：
- 如果localStorage中没有userId，自动生成并存储
- 所有API请求自动携带userId（通过请求头 `X-User-Id`）

## 🔄 数据流转

### 刷题流程
```
用户选择题型 → 选择答题模式 → 开始答题
  ↓
获取题目列表 (questionApi.getQuestionList)
  ↓
用户答题 → 保存答题记录 (recordApi.saveAnswerRecord)
  ↓
显示答案/解析 → 可选：AI辅助 (aiApi.ask)
  ↓
可选：保存笔记 (noteApi.saveNote)
  ↓
错题自动进入错题本
```

### 视频播放流程
```
选择课程 → 进入播放页
  ↓
加载视频 → 初始化播放器
  ↓
用户操作：
  - 发送弹幕 → WebSocket/HTTP推送
  - 开启AI字幕 → useWhisperSubtitle.start()
    ↓
    捕获音频 → 重采样 → Whisper识别 → 显示字幕
```
