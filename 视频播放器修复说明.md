## 🚀 使用说明

### 基本使用
1. 从课程列表选择章节，自动跳转到播放页
2. 视频自动加载，显示加载动画
3. 点击播放按钮或视频区域开始播放

### 控制功能
- **播放/暂停**：点击控制栏的播放按钮
- **进度控制**：拖动进度条或点击进度条
- **倍速播放**：选择倍速下拉菜单
- **全屏**：点击全屏按钮（F键）
- **静音**：点击静音按钮

### AI字幕
1. 点击"开启AI字幕"按钮
2. 等待WebSocket连接成功
3. 字幕实时显示在视频下方

### 弹幕功能
1. 在弹幕输入框输入内容
2. 按回车或点击"发送"按钮
3. 弹幕从右侧滑入显示

## ⚠️ 注意事项

### 1. **浏览器兼容性**
- 需要支持 HTML5 Video API
- 需要支持 Web Audio API（AI字幕功能）
- 需要支持 WebSocket（AI字幕功能）
- 建议使用 Chrome、Edge、Firefox 最新版本

### 2. **视频格式**
- 目前支持 MP4 格式
- 视频URL需要支持CORS
- 如果远程加载失败，会自动尝试Blob方式加载

### 3. **AI字幕功能**
- 需要后端WebSocket服务运行在 `ws://localhost:8088/gangwei/ws/asr`
- 需要浏览器支持Web Audio API
- 首次使用需要用户交互（点击）来激活AudioContext

### 4. **全屏功能**
- 不同浏览器使用不同的全屏API
- 代码已兼容主流浏览器（Chrome、Firefox、Safari、Edge）
