### 1. **修复全屏时倍速选择器无法展开的问题**

**问题**：全屏模式下，Element Plus 的 `el-select` 下拉菜单被遮挡，无法正常展开。

**修复方案**：
- 设置 `popper-append-to-body="false"`，让下拉菜单附加到当前元素
- 添加 `popper-class="video-speed-select"` 自定义类名
- 通过 CSS 设置全屏时的高 z-index（99999）

```vue
<el-select 
  v-model="playbackRate" 
  :popper-append-to-body="false"
  popper-class="video-speed-select"
>
```

```css
:fullscreen .video-speed-select {
  z-index: 99999 !important;
}
```

## 📝 代码实现细节

### 1. 键盘事件处理

```javascript
const handleKeyboard = (e) => {
  // 检测是否在输入框中
  const activeElement = document.activeElement;
  if (activeElement && (
    activeElement.tagName === 'INPUT' ||
    activeElement.tagName === 'TEXTAREA' ||
    activeElement.isContentEditable
  )) {
    // 在输入框中时，只处理空格键（如果不在输入框）
    if (e.code === 'Space' && activeElement.tagName === 'INPUT') {
      return;
    }
    if (e.code !== 'Space') {
      return;
    }
  }

  e.preventDefault();
  e.stopPropagation();

  // 根据按键执行相应操作
  switch (e.code) {
    case 'Space':
      togglePlay();
      break;
    // ... 其他快捷键
  }
};
```

### 2. 音量控制

```javascript
// 音量调节
const setVolume = (vol) => {
  volume.value = Math.max(0, Math.min(1, vol));
  videoEl.value.volume = volume.value;
  videoEl.value.muted = volume.value === 0;
  isMuted.value = volume.value === 0;
};

// 音量增加/减少
const increaseVolume = () => setVolume(volume.value + 0.1);
const decreaseVolume = () => setVolume(volume.value - 0.1);
```

### 3. 快进/快退

```javascript
// 快进/快退（秒数）
const seek = (seconds) => {
  const newTime = Math.max(0, Math.min(
    videoDuration.value, 
    videoEl.value.currentTime + seconds
  ));
  videoEl.value.currentTime = newTime;
  currentTime.value = newTime;
  currentProgress.value = newTime;
  showSeekHint(seconds > 0 ? `快进 ${Math.abs(seconds)}秒` : `快退 ${Math.abs(seconds)}秒`);
};
```

### 4. 快捷键提示显示

const showKeyboardHintMessage = (message) => {

## 🚀 使用说明

### 基本操作

1. **播放/暂停**：按空格键或 K 键
2. **快进/快退**：按左右方向键（每次5秒）
3. **音量调节**：按上下方向键
4. **全屏**：按 F 键
5. **静音**：按 M 键

### 快速跳转

- 按数字键 0-9 快速跳转到视频的 0%-90% 位置
- 例如：按 5 跳转到 50% 位置

### 查看帮助

- 点击控制面板中的"快捷键帮助"按钮
- 或按 I 键显示/隐藏快捷键列表

## ⚠️ 注意事项

### 1. 输入框冲突处理

- 当焦点在弹幕输入框时，大部分快捷键会被忽略
- 空格键在输入框中正常输入，不会触发播放/暂停
- 这是为了避免与输入操作冲突

### 2. 浏览器兼容性

- 所有快捷键功能在主流浏览器中正常工作
- 全屏功能已兼容 Chrome、Firefox、Safari、Edge

### 3. 性能考虑

- 快捷键事件监听在组件卸载时会自动清理
- 提示消息使用防抖机制，避免频繁显示
